このプロジェクトは「じゃんけん」アプリです。

通常の「自分の手」を選択後、「相手のランダムの手 (以下COMの手)」と戦い、結果を表示します。

その他の機能として
・1回目でプレイヤーが勝利した場合、次のCOMの手は「1回目で出した自分の手」に勝てる手を出します。
・1回目でプレイヤーが敗北した場合、COMは次に出す手を変更します。
・同じ手でCOMの手が連勝した場合は、自動で次の手を変えるようになっております。

下記の素材を編集した じゃんけんカードを使用しております。
